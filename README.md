# Projeto DOT_WELL
Ok, como diria Jack Estripador: "vamos por partes".

DOT_WELL é uma linguagem de programação criada por mim, inspirada em linguagens como Rust, C, Python e outras. Ela é projetada para ser chata e detalhista, exigindo que o programador defina tudo com precisão, o que pode ser útil para aprender conceitos de programação e lógica.

## Características Principais
- Tipagem Estática: Todas as variáveis devem ter seus tipos definidos explicitamente.
- Controle de Fluxo: Suporte a estruturas condicionais (Ainda não suporta loops).
- Macros: Suporte a macros para facilitar operações comuns.
